Hangman Game

Welcome to Hangman Game


Hangman game is a word-guessing game where the player tries to guess a hidden word by suggesting letters within a certain number of attempts. The game displays a series of underscores representing the letters of the hidden word, and the player guesses letters one at a time. If the guessed letter is present in the word, the corresponding underscores are replaced with the correct letter. If the guessed letter is not in the word, the player loses an attempt and part of a hangman figure is gradually drawn. The player wins if they guess the word correctly before running out of attempts, and loses if the hangman figure is completed before guessing the word.
